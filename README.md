# iMauzo UI (Meta Design System)

Bienvenue dans la documentation officielle de **iMauzo UI V1.0.0**, un framework de composants Vue 3 Premium architecturé sur Tailwind CSS V4 et DaisyUI.  
Pensé et conçu avec une précision maniaque, ce framework transpose de façon native l'authenticité et la rigueur du **Meta Business Suite** (Facebook).

---
